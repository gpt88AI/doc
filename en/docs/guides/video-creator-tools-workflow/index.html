6 个视频创作工具与工作流

整理 Excalidraw、Screen Studio、Rotato、Jacky Illustration、Jacky Motion、MD Publish 六个视频创作工具的定位、适用场景和组合工作流。

This page is accessible in English navigation, but the full body has not been translated yet. The original Chinese content is kept below for accuracy.
结论

这 6 个工具不是同一类软件。它们分别覆盖脚本、手绘图、录屏、设备动画、口播配图、口播动画和多平台发布。 真正高效的做法，是把它们串成一条内容生产线，而不是每条视频都从零剪辑。

核心思路

短视频创作的效率瓶颈通常不在“会不会剪辑”，而在素材生产和结构表达。知识类内容需要观众看懂， 产品类内容需要快速展示操作，口播类内容需要画面不断给信息，科技类内容需要设备动效提升质感。

这套工具组合的价值，是把内容生产拆成多个低成本环节：用 Markdown 写脚本，用手绘图降低理解成本， 用录屏替代复杂剪辑，用设备动画做包装，用自动配图和动画补足视觉，再用一键排版完成多平台分发。

6 个工具分别解决什么

工具定位适合内容关键价值
Excalidraw开源手绘画板，可装在 Obsidian 里使用。知识类、教程类、概念解释、方法论拆解。手绘感天然有真人痕迹，观众更愿意停留，比纯 AI 图更有温度。
Screen Studio智能录屏工具，自动缩放、背景、提词器和手机画面接入。产品演示、网页教程、App 操作、软件功能讲解。录完接近成品，减少后期剪辑时间。
Rotato设备动画工具，把截图放进 iPhone、MacBook、iPad 等设备模板。科技产品、SaaS、App、官网展示、发布会风格短片。几分钟做出设备旋转、开合、转场等科技感镜头，替代复杂 AE 制作。
Jacky Illustration口播脚本转结构化配图技能。观点口播、知识讲解、黑板风、涂鸦风、解释型内容。把口播稿自动拆成配图，不需要设计基础。
Jacky Motion口播稿转 HTML 动画和镜头节拍的工具。口播短视频、财经演播室、苹果科技风、知识动画。自动拆分镜头节奏，用动画承接口播信息密度。
MD PublishMarkdown 一键多平台排版工具。公众号、小红书、知识星球、社群长文和多平台分发。在 Obsidian 写完后自动适配平台格式，减少重复调版。

推荐工作流

video-production-flowtext
一条视频的推荐流程：
1. 在 Obsidian 里写 Markdown 脚本
2. 用 Excalidraw 画概念图、流程图或关键结构
3. 用 Jacky Illustration 把口播稿转成配图方案
4. 用 Jacky Motion 把口播稿拆成镜头节拍和 HTML 动画
5. 用 Screen Studio 录制产品演示、网页操作或手机画面
6. 用 Rotato 生成设备动效镜头，补足科技感开场和转场
7. 用 MD Publish 把脚本文案一键适配公众号、小红书等平台

这条流程的关键是先定脚本，再按信息类型选素材。需要解释概念时用 Excalidraw，需要演示操作时用 Screen Studio， 需要包装设备时用 Rotato，需要让口播更有画面时用 Jacky Illustration 或 Jacky Motion。

工具详解

Excalidraw：让知识内容有“人画过”的痕迹

Excalidraw 适合画关系图、流程图、系统结构、前后对比和概念草图。它的优势不是精致，而是“像人随手画出来的”。 对知识类内容来说，这种不完美反而会降低距离感，让观众更容易停留。

  • 适合放在视频开头，快速解释问题结构。
  • 适合和 Obsidian 搭配，把脚本、资料和图放在同一套笔记系统里。
  • 不要追求复杂插画，重点是让观众 3 秒内看懂关系。

Screen Studio：把录屏变成接近成品的镜头

Screen Studio 的优势是自动缩放、镜头跟随、渐变背景和提词器。它适合做产品教程、工具演示和网页操作， 可以把原本需要大量关键帧的录屏，直接变成更适合短视频观看的镜头。

  • 录制前先写好操作路径，减少返工。
  • 关键点击、输入、切换页面的位置要停顿 1 到 2 秒，方便自动缩放识别。
  • 如果是口播教程，可以用提词器保持节奏一致。

Rotato：快速补齐科技感包装

Rotato 适合把 App 截图、网页截图或产品界面放进设备模型，再生成旋转、开合、滑入等动画。 它最适合做开场、章节转场、产品发布风格镜头和社媒首屏素材。

  • 截图要先清理状态栏、通知、测试数据和隐私信息。
  • 设备动画不要堆太多，1 到 2 个关键镜头足够提升质感。
  • 复杂 UI 建议用慢速镜头，避免观众看不清画面。

Jacky Illustration：口播稿自动转配图

Jacky Illustration 适合把一段口播稿拆成结构化配图。它的价值是把抽象观点变成可视化节点， 例如涂鸦风、黑板风、手绘解释图或分步骤图解。

  • 输入脚本时要标清段落主题，不要把所有内容塞进一段。
  • 每张配图只表达一个观点，避免文字和元素过多。
  • 适合配合 Excalidraw 使用，一个负责即时草图，一个负责批量配图。

Jacky Motion：把口播拆成动画节拍

Jacky Motion 更适合动态内容。它会把口播稿拆成镜头节拍，并生成 HTML 动画方向。 如果你的内容是科技、财经、知识解说或方法论讲解，它可以替代大量手动动效设计。

  • 先把口播稿按 5 到 10 秒一段拆开，再交给工具处理。
  • 选择风格前先看内容类型：苹果科技风适合产品，财经演播室适合数据与观点。
  • 动画只是辅助，核心仍然是信息节奏和旁白结构。

MD Publish：让一条内容同时服务多个平台

MD Publish 适合把 Obsidian 或 Markdown 里的内容自动转成不同平台格式。它解决的是最后一公里： 同一篇脚本既能变成视频文案，也能变成公众号文章、小红书图文和社群长文。

  • 写作时用清晰标题、短段落和列表，方便多平台转换。
  • 视频脚本可以保留“镜头说明”和“口播文案”两个区块。
  • 发布前仍要人工检查平台敏感词、图片比例和标题长度。

组合方案

quick-stacktext
知识类视频 / 教程类视频：
Excalidraw + Screen Studio + MD Publish

产品演示 / 科技感短视频：
Screen Studio + Rotato + Jacky Motion

口播号 / 观点类内容：
Jacky Illustration + Jacky Motion + MD Publish

AI 视频 / 分镜类内容：
Excalidraw + Jacky Motion + video-generation-skills

  选 Excalidraw

  选 Screen Studio

需要 iPhone / MacBook / iPad 科技动画
  选 Rotato

  选 Jacky Illustration

  选 Jacky Motion

常见误区

不要把所有工具都用在同一条视频里。工具越多，素材风格越容易散。更稳定的做法是先确定内容类型， 再选择 2 到 3 个核心工具形成固定模板。
