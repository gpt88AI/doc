每次請求使用 Bearer 認證。請將 Key 儲存在環境變數或密鑰管理器中，並以控制台的即時資訊為準。

  • 不要在瀏覽器程式碼或公開倉庫暴露 API Key。
  • 發現洩漏後立即在控制台撤銷並輪換。
  • 價格、配額、模型和路線以目前控制台為準。
