gemini-2.5-flash-image

Google gemini-2.5-flash-image · IMAGE 模型

在 gpt88.cc 获取 API Key快速开始

模型概览

  • 这是一个图像生成模型，适合把文本提示词转成静态图片或视觉素材。
  • 它属于 Google Gemini 路线，通常更适合多模态理解、图像输入或综合推理任务。
  • 名称中的 flash / mini 变体通常更偏低延迟、高吞吐或轻量任务。

适用场景

  • 需要把文案或提示词转换成图片素材时
  • 做海报、电商主图、品牌概念图或社媒配图时
  • 先用一条最小 prompt 验证输出风格，再扩展到批量生成时

接入说明

  • 图片与视频模型统一使用图片与多媒体 Base URL https://img.gpt88.cc。
  • 第一次接入时，建议先用 /v1/images/generations 验证 API Key、模型名 gemini-2.5-flash-image 和当前线路是否匹配。
  • 媒体类模型通常先验证上传 / 返回结构，再扩展到正式的素材或媒体管线。

使用提醒

  • flash / mini 变体更适合轻量任务；面对复杂长文档或高质量推理时，可能需要升级到更高阶模型。
  • 价格、限速、SLA、上下文长度、是否开放多模态等动态值以 gpt88.cc 控制台为准。
  • 如果请求延迟偏高或连接不稳定，请先检查 API Key、模型、endpoint 和请求格式。

错误与排查

  • 401：检查 Authorization 是否为 Bearer <API_KEY>，以及 Key 是否仍有效。

  • 5xx：保持统一 Base URL 重试，并记录请求 ID 方便排障。

参考图生成

上传参考图文件，在保留主体或风格的基础上继续生成。

import requests

with open("reference.png", "rb") as f:
    resp = requests.post(
        "https://img.gpt88.cc/v1/images/edits",
        headers={
            "Authorization": "Bearer YOUR_GPT88_API_KEY",
        },
        files={
            "image": f,
        },
        data={
            "model": "gemini-2.5-flash-image",
            "prompt": "保留参考图主体，改成极简 API 文档站封面风格",
            "size": "1536x1024",
            "quality": "high",
        },
    )
print(resp.json())
